# Project boundary

`modular-body-config-lab` is a public notebook for modular CAD configuration studies and replay-oriented simulation inputs.

## In scope

- CAD configuration learning
- modular body and slot vocabulary
- placeholder module naming
- configuration tables
- export discipline
- coarse mass, power, cooling, and geometry classes
- simple simulation-input preparation
- public learning notes

## Out of scope

- weapons
- tactics
- evasion
- stealth
- defense simulation
- controlled or sensitive data
- operational system design
- flight-performance claims
- CFD claims
- durability or structural-load claims
- real vehicle integration

## Working rule

Prefer placeholder geometry and configuration data over detailed mechanism design.

A useful artifact here should answer:

```text
What is the module?
Where does it attach?
What coarse class does it belong to?
What would a later simulator need to know?
```

A non-useful artifact here tries to answer operational behavior, mission design, or performance claims.
