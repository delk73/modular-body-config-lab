# Project Boundary

This repository is a public CAD and configuration learning notebook.

## In scope

- CAD learning notes
- modular placeholder assemblies
- configuration tables
- geometry envelope notes
- mass, power, and cooling classes
- export notes
- simulation-input preparation

## Out of scope

- weapons modeling
- targeting
- evasion
- stealth optimization
- operational tactics
- real-world platform design claims
- defense simulation
- controlled or sensitive data

## Working rule

Keep public artifacts abstract, educational, and configuration-oriented.
