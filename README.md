# Modular Body Configuration Lab

Public notebook for learning CAD configuration workflows and producing simple configuration artifacts that can later feed replay-oriented simulation tools.

## Purpose

This repository tracks a CAD/simulation learning path focused on modular assemblies, configuration tables, geometry envelopes, and exported simulation inputs.

The first objective is modest:

- create a placeholder base body
- create a module slot
- create interchangeable placeholder modules
- define configuration metadata
- export a simple configuration table

## Boundary

This repository is for CAD learning, modular assembly studies, and simulation-input preparation.

It does not model weapons, tactics, evasion, targeting, stealth, or operational systems.

## Current study

`modular_platform_config_v0`

A minimal placeholder assembly with:

- base body
- module slot
- blank module
- sensor placeholder
- relay placeholder
- lighting placeholder
- exported configuration table

## Downstream use

Configuration tables from this repository may later be consumed by private replay or simulation tooling.
