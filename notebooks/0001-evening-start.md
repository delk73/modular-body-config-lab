# 0001 — Evening start

## Goal

Create the smallest useful modular CAD/configuration exercise.

## Exercise

Sketch or model a simple base body with one module slot and several placeholder module concepts:

- `base_body_v0`
- `module_slot_v0`
- `blank_module_v0`
- `sensor_placeholder_v0`
- `relay_placeholder_v0`
- `lighting_placeholder_v0`

## What to learn

- how the base body and module slot are named
- how placeholder modules share an interface
- how geometry class differs from module role
- how a configuration table can represent the design before detailed CAD export

## Success condition

A successful session produces one of these:

- a rough CAD sketch
- a screenshot or exported neutral placeholder
- a configuration-table update
- a short learning-log entry

It does not need to produce a finished model.

## Stop condition

Stop before entering CFD, durability, flight dynamics, or detailed simulation. This notebook is for configuration literacy first.
